Linux新机备份

以备换机器使用

Linux

  1. 安装zsh 

    $ apt update
$ apt install zsh -y

  2. 更改默认shell

    $ chsh -s /bin/zsh
# 查看当前shell 
# echo $SHELL
# 查看系统shells
# cat /etc/shells

  3. 解决chsh(若有): PAM: Authentication failure，直接修改用户配置，把bash改成zsh

    $ sudo vim /etc/passwd
  root:x:0:0:root:/root:/usr/bin/bash

  4. 安装git

    $ apt install git -y

  5. 安装oh-my-zsh

    $ sh -c "$(curl -fsSL 
https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

  • 卸载oh-my-zsh
    卸载oh-my-zsh, 只需要执行uninstall_oh_my_zsh zsh， 从命令行运行. 这将删除本身和恢复你以前的bash或者zsh配置

    $ uninstall_oh_my_zsh zsh
或者
$ sh -c "$(curl -fsSL 
https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/uninstall.sh)"
